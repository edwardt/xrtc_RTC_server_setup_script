#xrtc_RTC_server_setup_script
This script creates configuration files used/read by processes/components running on the RTC server

##Configuration files
- prosody XMPP
- Jitsi Components
  - Jicofo (FOCUS)
  - Videobridge

## Running the script
```sh
# python create_rtc_config_files.py  config.ini 
```
